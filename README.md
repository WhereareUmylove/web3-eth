
## Project setup（安装库）
```
npm install
```

### Compiles and hot-reloads for development（启动开发版本）
```
npm run serve
```

### Compiles and minifies for production version(正式版本打包编译)
```
npm run build
```

### Compiles and minifies for test version(测试版本打包编译)
```
npm run test
```
